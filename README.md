## 介绍

自动化的自媒体内容制造机 自动运维自媒体 无人值守创作直播 全平台可运行 做一个有思想的复读机 有金饭碗的ebegger

计划将ChatGPT类模型接入本项目 构造提示词集合 流水线式工作 (可以创建以前利用传统程序难以做到的事情 比如编写剧本 追踪视频来源 提取还原URL 反馈学习等等)
